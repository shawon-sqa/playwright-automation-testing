# Playwright Automation Testing

Professional end-to-end automation testing project built with Playwright.

## Project Overview

This project demonstrates real-world QA automation scenarios including:

- Login Testing
- UI Validation
- Functional Testing
- Browser Automation
- Assertion Validation

## Tools & Technologies

- Playwright
- JavaScript
- Node.js
- VS Code
- Git
- GitHub

## Test Scenarios

### Valid Login Test

- Navigate to login page
- Enter valid credentials
- Click login button
- Verify successful login
